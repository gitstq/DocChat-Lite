# DocChat-Lite

A lightweight, zero-dependency document knowledge base chat engine. Pure frontend, offline-first web application.

## Features

- Document management (TXT, MD, JSON, CSV, HTML)
- TF-IDF full-text search with Chinese support
- Chat with LLM integration (OpenAI-compatible API)
- Dark/Light theme
- Responsive design
- IndexedDB storage (offline-first)
- Zero npm dependencies

## Usage

Open `index.html` in a modern browser. No build step required.
